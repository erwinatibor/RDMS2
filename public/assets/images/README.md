# assets/images/

Place real photography and image assets here.

## Recommended Images to Add

| Filename                  | Usage                          | Recommended Size  |
|---------------------------|--------------------------------|-------------------|
| `hero-bg.jpg`             | Hero section background        | 1920×1080px       |
| `mobile-clinic.jpg`       | Programs / mobile clinic card  | 800×600px         |
| `dental-workshop.jpg`     | Services / workshops card      | 800×600px         |
| `school-outreach.jpg`     | Services / school programs     | 800×600px         |
| `team.jpg`                | About section                  | 900×600px         |
| `community.jpg`           | General community outreach     | 800×600px         |
| `og-image.jpg`            | Social sharing preview         | 1200×630px        |

## Image Guidelines

- Compress all JPGs to ≤200KB using TinyPNG or Squoosh
- Preferred format: WebP for web (with JPEG fallback)
- Add `width` and `height` attributes to prevent layout shift
- Use `loading="lazy"` for below-fold images
- Alt text must describe the content, not just say "photo"
